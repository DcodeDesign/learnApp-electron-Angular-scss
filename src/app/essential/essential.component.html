1. Init Projet Angular :

1.1 NodeJs

Runtime JavaScript basé sur le moteur JavaScript V8 de Chrome.

Download : https://nodejs.org/en/

  
    > node -v

1.2 NPM

Gestionnaire de packages Node

SiteWeb : npmjs.com

  
  > sudo npm install npm -g
  > npm -v

1.3 Angular CLI

interface de ligne de commande pour Angular
  
    > npm install -g @angular/cli

1.4 Create Angular projet

  
  > ng new [nameProjet]
  > cd firstApp
  > ng serve

ng serve ~> compile le projet et démarre un serveur de développement http://localhost/4200

2. Description Folders & Files Angular

2.1 Folders

  • **src** ~> dossier regroupant l'enssemble des fichier des l'application
  • **assets** ~> on y trouve les images, css, js additionnel
  • node_modules ~> dossier regroupant les dépendances du projet
  • e2e ~> permet l'execution de test

2.2 Files

  • index.html ~> point d'entrée de l'application
  • main.js ~> point entrée pour le typeScript
  • style.css ~> style global à l'application
  • package.json ~> liste les dépendances utilisés dans le projet
  • package-lock.json ~> liste les dépendances utilisés dans le projet avec leurs versions exactes
  • angular.json ~> fichier de configuration d'angular et à la CLI
